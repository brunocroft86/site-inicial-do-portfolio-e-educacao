## 🎯 Sobre o Projeto

Uma "Single Page Application" (Página Única) moderna, focada em conversão e autoridade. O layout foi construído com um tema escuro (Dark Mode) e detalhes em neon, mesclando a pegada tecnológica do desenvolvimento de software com a responsabilidade social e educacional do ensino da tecnologia.

### O que você encontra no site:
* **Serviços:** Criação de Sites, Sistemas Web e E-commerces.
* **Portfólio:** Projetos reais entregues para clientes (Açaí da Preta, Hortifruti Coisa Linda, etc).
* **Aulas Particulares:** Mentoria e aulas 1 a 1 de Programação, Informática e Pacote Office.
* **Impacto Social:** Destaque para o projeto itinerante "Mulheres Conectadas", focado em inclusão digital.

---

## ✨ Funcionalidades e UI/UX

* **Animações Fluidas:** Utilização da biblioteca GSAP (ScrollTrigger) para revelar elementos ao rolar a página (Fade-up effects).
* **Totalmente Responsivo:** O layout se adapta perfeitamente a smartphones, tablets e desktops.
* **Otimização de SEO:** Meta tags configuradas (Open Graph) para indexação no Google e compartilhamento bonito em redes sociais (WhatsApp, LinkedIn).
* **Call to Action (CTA):** Botão flutuante integrado diretamente com a API do WhatsApp para contato rápido.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído sem frameworks pesados no front-end, focando em performance e carregamento rápido:

* **HTML5** - Estruturação semântica e SEO.
* **CSS3** - Estilização customizada com variáveis (CSS Variables) e Grid/Flexbox.
* **JavaScript** - Lógica de interação.
* **[GSAP](https://greensock.com/gsap/)** - Biblioteca de animações de alto desempenho.
* **[FontAwesome](https://fontawesome.com/)** - Ícones vetoriais.
* **Google Fonts** - Tipografia (Poppins).
